\section{Introduction}
This report summarizes a reproducible training pipeline for controlling an agent in MiniGrid EmptyEnv with NanoVLM, comparing supervised fine-tuning (SFT) and two GRPO variants.

\section{Method}
We train a policy to output action tokens from RGB observations and mission text. GRPO uses grouped rollouts and normalized advantages within each group.

\section{Setup}
Default environment is \texttt{MiniGrid-Empty-8x8-v0} with \texttt{RGBImgObsWrapper}. The model starts from \texttt{lusxvr/nanoVLM-222M} and uses PEFT-lite updates.
